# curator-pwa

Система куратора для керування інформацією про студентів та їхніх батьків.

## Зміни в базі даних

У таблиці `parents` поле `type` тепер використовує значення:
- `mother` - Мати
- `father` - Батько

Попередні значення `мати`, `батько`, `опікун` були замінені на англійські аналоги для стандартизації.
